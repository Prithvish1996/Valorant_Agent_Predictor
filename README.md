Valorant_Agent_Predictor
==============================

A machine learning project to predict Valorant agents

## Setup

### 1. Install Conda

Download and install [Anaconda](https://www.anaconda.com/download) or [Miniconda](https://docs.conda.io/en/latest/miniconda.html)

### 2. Create the conda environment

```bash
conda env create -f environment.yml
```

### 3. Activate the environment

**Windows (Command Prompt/PowerShell):**
```bash
conda activate valorant_predictor
```

**macOS/Linux:**
```bash
conda activate valorant_predictor
```
